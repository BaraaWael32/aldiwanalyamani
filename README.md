
# مشروع الديوان اليمني

## للتجربه : aldiwanalyamani.vercel.app

هذا المشروع عبارة عن موقع ويب مبني باستخدام React وVite، ويهدف إلى تقديم واجهة حديثة وسريعة مع دعم HMR (تحديث الوحدة السريع) وتهيئة ESLint لضمان جودة الكود.

## مميزات المشروع

- استخدام React لإنشاء واجهات المستخدم التفاعلية.
- الاعتماد على Vite لتسريع عملية التطوير والبناء.
- دعم Tailwind CSS لتصميم واجهات عصرية وسهلة التخصيص.
- تنظيم الكود في مجلدات (components, pages, assets) لسهولة الصيانة والتطوير.
- إعداد ESLint لفحص الكود وتحسين جودته.

## هيكلية المشروع

- src/
  - components/ : مكونات الواجهة مثل Navbar وFooter وScrollToTop
  - pages/ : الصفحات الرئيسية مثل الرئيسية، من نحن، الخدمات، المعرض، التواصل
  - assets/ : الصور والملفات الثابتة
  - App.jsx : نقطة البداية لتجميع المكونات
  - main.jsx : ملف الدخول الرئيسي للتطبيق

## بدء التشغيل

لتشغيل المشروع محلياً:

1. تثبيت الحزم:
	```bash
	npm install
	```
2. تشغيل الخادم المحلي:
	```bash
	npm run dev
	```

## البناء للإنتاج

لبناء نسخة إنتاجية:
```bash
npm run build
```

## معلومات إضافية

- تم إعداد ESLint وTailwind CSS وPostCSS مسبقاً.
- يمكن تعديل الإعدادات من خلال ملفات التهيئة مثل tailwind.config.js وvite.config.js.

لأي استفسار أو مساهمة، يرجى التواصل مع فريق التطوير.
